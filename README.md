## Email Checker
Um projeto de classificação de e-mails (Produtivo / Improdutivo) usando **Transformers** (Hugging Face) e **Flask** para servir a aplicação via web.

### Funcionalidades

Classificação de texto de e-mails entre as duas classes (Produtivo/Improdutivo).

Classificação de arquivos anexados (PDF, TXT, EML).

Servidor web baseado em Flask.

### Para executar localmente

1. Clone o repositório

2. Instale as dependências

pip install -r requirements.txt

3. Execute a aplicação no terminal

python app.py

4. Abra o navegador

Insira o http://127.0.0.1:7860
